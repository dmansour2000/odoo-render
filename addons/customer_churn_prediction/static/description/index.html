Customer Churn Prediction

Customer churn prediction module for Odoo. This solution analyzes customer behavior patterns to identify which customers are at risk of leaving your business before they actually do.


Key Features

  • Intelligent Churn Detection: Uses machine learning to identify patterns in customer behavior that indicate potential churn.
  • Multi-Factor Analysis: Considers numerous factors including:
    • Purchase frequency and recency
    • Order values and total spend
    • Customer lifetime
    • Payment patterns
    • Account activity
  • Visual Indicators: Adds warning badges to at-risk customers in list and kanban views
  • Retention Button: Adds a direct "Retention Required" button to quickly create opportunities for at-risk customers
  • Automated Updates: Daily scheduled task refreshes churn predictions automatically

Technical Details

  • Uses Random Forest classifier class
  • Stores model data efficiently using binary attachments
  • Customers who ordered before but not in the last 6 months are considered churned.
  • To process the train model must be at least 100 active records (customers & orders).

Installation

  • Install the required Python dependencies: pip install pandas numpy scikit-learn
  • Install the module through Odoo's Apps menu "customer_churn_prediction"
  • Go to CRM > Configuration > Churn Prediction Models and click "Train Model"
